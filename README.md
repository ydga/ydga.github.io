# Starter Kit

**Pick a look → download a themed shadcn repo and a matching Figma library in one step.** No design system expertise required.

**Status:** Planning — see [`docs/ARCHITECTURE.md`](./docs/ARCHITECTURE.md).

## What you get (one download)

| Part            | What it is                                                     |
| --------------- | -------------------------------------------------------------- |
| **`code/`**     | Next.js + shadcn, 12 core components, light/dark, ready to run |
| **`figma/`**    | Matching Figma library + variables (same theme as code)        |
| **`make-kit/`** | Figma Make guidelines so AI prototypes use your system         |

## How it works

1. Choose a preset (Minimal, SaaS, Warm, …)
2. Optionally tweak brand color, font, corner radius
3. Preview components in the browser
4. Download one ZIP → run code, duplicate Figma, optional Make kit setup

No login. Session saved in the browser — [download before leaving](./docs/SESSION.md).

## Hosting

- **Now:** GitHub Pages (`ydga.github.io`)
- **Later:** Vercel optional
